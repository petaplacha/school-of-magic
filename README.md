# School of Magic

Tento projekt je React hra vytvořená pomocí Vite. Build je připraven pro GitHub Pages ve složce `docs`.

## Vývoj

```
npm install
npm run dev
```

## Build a nasazení na GitHub Pages

1. Proveďte build:
   ```
   npm run build
   ```
2. Nasazení na GitHub Pages:
   ```
   npm run deploy
   ```

Výstupní stránka bude dostupná na https://<uživatel>.github.io/school-of-magic-git/

---

Kód hlavní hry je převáděn z původního `main.js` do komponent v `src/`.
